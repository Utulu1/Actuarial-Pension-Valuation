# Actuarial Pension Valuation

## Project Overview

This project demonstrates basic actuarial pension valuation concepts using discounted pension benefits.

The analysis explores retirement ages, pension payments, and present value calculations used in actuarial pension work.

## Objectives

* Analyse pension benefits
* Apply present value calculations
* Explore pension liabilities
* Demonstrate actuarial valuation concepts
